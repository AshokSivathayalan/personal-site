Predicting Canadian wildfire ignition causes

2026

XGBoost classifier with SHAP analysis on the CNFDB dataset (~374K fires)

Trained an XGBoost classifier to predict ignition cause (lightning vs. human and finer subcategories) across the Canadian National Fire Database, around 374,000 fires spanning decades. Engineered features from spatial, temporal, and meteorological signals, handled the substantial class imbalance, and used SHAP values to surface which features drove predictions in each regime. Wrote up the analysis as a LaTeX report and built a large-format academic poster programmatically with PptxGenJS, presented at Data Day 2026.

Bird species image classifier

2025

Fine-tuned ResNet50 for 8 North American bird species, with Grad-CAM explainability

Fine-tuned a ResNet50 (pretrained on ImageNet) on ~14,700 iNaturalist images across 8 North American bird species, applying class weights to handle imbalance across species. Ran a hyperparameter sweep over learning rate, batch size, and optimizer; the best configuration (Adam, LR=0.01, batch size 64) reached 81.61% validation accuracy. Built a Gradio interface with Grad-CAM visualizations that highlight which regions of the image the model is using for its prediction, and deployed it as a live demo on Hugging Face Spaces. Also wrote the data-collection and train/val/test split pipeline against the iNaturalist API.

python pytorch resnet transfer-learning grad-cam gradio
